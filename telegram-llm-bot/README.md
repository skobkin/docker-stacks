# Telegram LLM bot

[Telegram LLM bot](https://git.skobk.in/skobkin/telegram-ollama-reply-bot) is a simple Telegram bot which uses LLM's to participate in discussions or make summarizations of the text at the provided source.

## Prerequisites

This service requires the `ai-tools` Docker network. Please follow the [network configuration guide](../_docs/ai_tools_network.md) before starting the service.

For general setup instructions, please refer to the [root README](../README.md).
